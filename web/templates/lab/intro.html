개요

문장 분리 및 군더더기 제거
문장 단위를 구별해서 반환합니다.
형태소 분석
문장 단위를 구별한 뒤, 각각의 형태소 상세내역을 분석해서 반환합니다.
저널리즘 가치 점수화
언론재단 뉴스트러스트 위원회에서 분석한 알고리즘에 기반한 평가점수를 반환합니다.
통합 평가
앞선 분석 내용들을 모두 종합해서 반환합니다.

1. 형태소 분석 (api/postag)

curl -X POST \
http://ntadm.newstrust.kr/api/postag \
-H 'content-type: multipart/form-data' \
-F 'content=이 부분에 테스트할 문단을 입력합니다.'

key header / field note
content-type header multipart/form-data
content field 분석을 요청할 내용

2. 계량 요인 추출 (api/metric)

5. 통합 평가 (api/integrate)

curl -X POST \
http://ntadm.newstrust.kr/api/trust \
-H 'content-type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F 'title=기사 제목'
-F 'content=기사 본문..'
-F 'mediaType=신문'
-F 'category=사회'

curl -X POST \
http://ntadm.newstrust.kr/api/integrate \
-H 'content-type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F 'title=기사 제목'
-F 'content=기사 본문..'
-F 'mediaType=신문'
-F 'category=사회'

key header / field note
content-type header multipart/form-data
title field 기사 제목
content field 형태소 분석을 요청할 내용
mediaType field (언론사 유형) 신문 / 방송
category field (카테고리) 정치 / 경제 / 사회 / 국제 / 문화 예술 / IT 과학 / 라이프스타일 / 스포츠 / 교육 / 연예
